Project Overview
เว็บไซต์ส่วนตัวแบบ Static จำนวน 8 หน้า พัฒนาด้วย HTML5 และ CSS
สำหรับแนะนำตัว แสดงประวัติการศึกษา ทักษะ ผลงานการบ้าน CV
แบบฟอร์มติดต่อ และแกลเลอรีรูปภาพ

Main Features
- เว็บไซต์อย่างน้อย 8 หน้า
- Navigation Menu ใช้งานได้ทุกหน้า
- Responsive Layout ด้วย Flexbox และ Grid
- Portfolio สำหรับ HW2 ถึง HW7
- ปุ่ม View สำหรับเปิดดูภาพผลงาน
- ปุ่ม Download สำหรับดาวน์โหลดไฟล์งาน ZIP
- Contact Form
- CV / Resume
- รูปภาพนักศึกษาและรูปผลงาน
- ใช้งานบน IIS ผ่าน port 8080
- เตรียมเผยแพร่บน Cloud
